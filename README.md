# Fitness Sport Center

A modern and responsive fitness landing website built with React and Vite. The project highlights key sections such as services, trainers, memberships, contact, and more for a complete gym/sports center showcase.

## Tech Stack

- React
- Vite
- Tailwind CSS (v4)
- React Icons
- ESLint

## Setup Instructions

### 1) Clone the repository

```bash
git clone https://github.com/rukshanvishwa/fitness-sport-center.git
cd fitness-sport-center
```

### 2) Install dependencies

```bash
npm install
```

### 3) Run in development mode

```bash
npm run dev
```

### 4) Build for production

```bash
npm run build
```

### 5) Preview production build

```bash
npm run preview
```
